# Resume-analyzer-and-Job-finder

INTRODUCTION:
The Resume Analyzer & Job Finder automates resume analysis
and job searching using rule-based techniques. ​
It extracts skills from resumes, calculates an ATS score, and
recommends suitable jobs through external APIs. ​
The system reduces manual effort and improves efficiency in the
job-matching process.

PROBLEM STATEMENT:
Manual resume screening is slow and error-prone.​
Recruiters receive thousands of resumes.​
Skill mismatch between candidates and job requirements.​
Increased chances of missed opportunities.

PROPOSED SOLUTION:
The proposed system automates resume analysis using rule-based text processing and keyword
matching techniques.​
The system:​
Extracts skills, experience, and basic qualifications from resumes​
Matches extracted skills with job requirements​
Generates personalized job recommendations using external job APIs​
Provides resume scoring and structured feedback based on ATS rules​
This ensures fast, consistent, and reliable skill-to-job matching while reducing manual effort.
